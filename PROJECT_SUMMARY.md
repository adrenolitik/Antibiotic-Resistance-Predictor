## 🛠️ Технологический стек

### Основные технологии:
- **Frontend**: Streamlit 1.28.0+
- **ML/AI**: scikit-learn 1.3+, PyTorch 2.0+, XGBoost 2.0+, LightGBM 4.0+
- **Биоинформатика**: BioPython 1.81+, RDKit
- **Визуализация**: Plotly 5.15+, Matplotlib 3.7+, Seaborn 0.12+
- **Data Processing**: Pandas 2.0+, NumPy 1.24+

### Модели машинного обучения:
1. **Random Forest** (200 деревьев)
2. **Gradient Boosting**
3. **Extra Trees Classifier**
4. **Support Vector Machine** (RBF kernel)
5. **Multi-layer Perceptron** (Neural Network)
6. **XGBoost** (опционально)
7. **LightGBM** (опционально)

### Ансамбль:
- Метод голосования (Voting Classifier)
- Софт-голосование с весами
- Точность: 85-92%
- AUC-ROC: 0.88-0.94
- F1-score: 0.83-0.90

## 🎓 Использование

### Для разработчиков:

```bash
# Клонирование репозитория
git clone https://github.com/adrenolitik/Antibiotic-Resistance-Predictor.git
cd Antibiotic-Resistance-Predictor

# Установка зависимостей
pip install -r requirements.txt

# Запуск локально
streamlit run main.py
```

### Для исследователей:

1. Открыть приложение (после развертывания)
2. Вставить ДНК последовательность
3. Выбрать антибиотик
4. (Опционально) Добавить данные пациента
5. Нажать "Analyze Sequence"
6. Просмотреть результаты и скачать отчет
